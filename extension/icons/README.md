# 图标文件

这个目录包含Chrome扩展所需的图标文件。

## 图标尺寸

- `icon16.png` - 16x16 像素，用于扩展管理页面
- `icon32.png` - 32x32 像素，用于Windows等系统
- `icon48.png` - 48x48 像素，用于扩展管理页面
- `icon128.png` - 128x128 像素，用于Chrome Web Store

## 图标设计

图标应该体现以下特点：
- 折叠/展开的概念
- AI聊天的主题
- 简洁现代的设计风格
- 支持深色和浅色主题

## 临时解决方案

在开发阶段，可以使用简单的占位符图标。以下是一些在线工具可以生成图标：

1. **Favicon.io**: https://favicon.io/
2. **Icon Generator**: https://icon-generator.net/
3. **Canva**: https://www.canva.com/create/icons/

## 建议的设计元素

- 使用箭头（上/下）表示折叠和展开
- 可以使用对话气泡或聊天气泡作为背景
- 主色调建议使用蓝色或紫色渐变
- 确保在小尺寸下仍然清晰可辨

## 版权说明

确保使用的图标不侵犯版权，可以使用：
- 自行设计的图标
- 开源图标库（如Material Icons、Font Awesome等）
- 购买商业许可的图标

## 文件格式

- 使用PNG格式以支持透明背景
- 确保文件大小合理（每个图标不超过50KB）
- 使用适当的压缩以平衡质量和文件大小
